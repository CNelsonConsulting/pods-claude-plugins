# Product Manager Plugin

Initial PM plugin scaffold for Claude cowork workflows.

## Layout

- `.claude-plugin/plugin.json`: plugin metadata
- `.mcp.json`: local MCP configuration starter
- `commands/*.md`: reusable PM command prompts
- `skills/*/SKILL.md`: PM operating skills

## Included command set

- `write-spec`
- `roadmap-update`
- `stakeholder-update`
- `synthesize-research`
- `competitive-brief`
- `metrics-review`
